#1 📱 APLIKASI SMARTPHONE ← PALING PENTING!
   Masalah: Sekarang Cendekia cuma bisa di browser (Chrome, Firefox)
   Solusi: Bikin aplikasi Android/iPhone biar mudah dibuka
   Impact: ⭐⭐⭐⭐⭐ (Mahasiswa senang banget)

#2 📧 NOTIFIKASI EMAIL
   Masalah: Kalo ada tugas baru / pengumuman, mahasiswa ga diberitahu
   Solusi: Otomatis kirim email ke mahasiswa kalo ada hal penting
   Impact: ⭐⭐⭐⭐⭐ (Students ga lagi kelewatan deadline)

#3 📊 LAPORAN & ANALYTICS
   Masalah: Guru ga bisa liat grafik / statistik siswa
   Solusi: Dashboard buat guru liat: siapa yg sering bolos, rata2 nilai, etc
   Impact: ⭐⭐⭐⭐ (Teachers bisa track progress)

#4 🔄 REAL-TIME UPDATE (Live)
   Masalah: Kalo presensi dibuka, harus refresh browser biar tau
   Solusi: Automatic update tanpa refresh (seperti chat WhatsApp)
   Impact: ⭐⭐⭐ (Better UX, tapi overkill untuk sekarang)

#5 🔍 PENCARIAN GLOBAL
   Masalah: Mau cari file/materi harus ke setiap kelas satu-satu
   Solusi: Tambah search box di navbar, tinggal ketik keyword
   Impact: ⭐⭐⭐ (Users bisa cari cepat)

#6 ⭐ PENILAIAN RUBRIK (Transparent Grading)
   Masalah: Nilai cuma 0-100, siswa ga tahu kenapa dapet nilai itu
   Solusi: Breakdown nilai: "5 poin struktur, 3 poin isi, 2 poin format"
   Impact: ⭐⭐⭐ (Penilaian lebih adil & jelas)

#7 🔐 KEAMANAN 2FA (Two-Factor Auth)
   Masalah: Akun guru/admin hanya pakai password, kurang aman
   Solusi: Tambah layer kedua: password + smartphone verification
   Impact: ⭐⭐⭐ (Security lebih baik)

#8 ⚡ PERFORMANCE TUNE
   Masalah: Kalo 1000 siswa login bersamaan, aplikasi lemot/error
   Solusi: Optimize database + tambah cache
   Impact: ⭐⭐⭐ (Aplikasi cepat & stable)

#9 🧪 TESTING & QA
   Masalah: Developer takut ada bug / fitur lama rusak
   Solusi: Buat automated tests
   Impact: ⭐⭐ (Better code quality, Developer senang)
