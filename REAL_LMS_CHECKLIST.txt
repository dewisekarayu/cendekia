#1 ❌ QUIZ / TEST SYSTEM ← PALING PENTING!!!
   Ada di: Canvas, Blackboard, Moodle, Google Classroom
   Cendekia: TIDAK ADA
   
   Contoh di Canvas:
   - Guru buat multiple choice, essay, fill-blank questions
   - Auto-grade objective questions (instant hasil)
   - Question bank (reuse questions di berbagai kuis)
   - Randomized questions per siswa (biar ga cheating)
   - Time limit per kuis
   - Detailed analytics: "80% siswa salah di soal #3"
   
   Analogi:
   - Canvas: Seperti punya automated test system (UNBK)
   - Cendekia: Ga punya, guru harus manual grade di paper
   
   IMPACT: Teachers harus bikin & grade test manual.
           Students ga bisa practice online.
           NO AUTO-GRADING!
           
   Timeline: 3-4 minggu
   ⭐ PRIORITY #1 BUAT BIKIN REAL LMS

─────────────────────────────────────────────────────────────────────────────

#2 ❌ COURSE STRUCTURE / LEARNING PATH
   Ada di: Canvas, Moodle
   Cendekia: Semua materi campur
   
   Contoh di Canvas:
   - Module 1: Intro (video + reading)
   - Module 2: Practice (assignments) - unlock jika Module 1 selesai
   - Module 3: Test
   - Progress bar: "Student udah 60% selesai course ini"
   
   Cendekia: Materials, assignments, forum, semua di dashboard tanpa urutan.
   
   Analogi:
   - Canvas: Seperti buku dengan chapter (Ch1 → Ch2 → Ch3)
   - Cendekia: Tumpukan kertas, students sorting sendiri
   
   IMPACT: Students bingung mulai dari mana.
           Teachers ga bisa enforce learning flow.
   
   Timeline: 2 minggu

─────────────────────────────────────────────────────────────────────────────

#3 ❌ ADVANCED GRADEBOOK
   Ada di: Canvas, Blackboard, Moodle
   Cendekia: Simple 0-100 grades
   
   Contoh di Canvas:
   - Weighted categories: "Quiz 30%, Assignment 40%, Attendance 30%"
   - Auto-calculate final grade
   - Grade trends: Grafik showing "Your grade going UP/DOWN"
   - What-if: "Kalo UAS dapet 90, final grade berapa?"
   - Compare dengan class: "You got 85, class avg 78"
   
   Cendekia: Just input 85, 90, 95. No calculation. No trend. Nothing.
   
   IMPACT: Teachers pusing hitung weighted grades.
           Students ga bisa liat progress trend.
   
   Timeline: 2 minggu

─────────────────────────────────────────────────────────────────────────────

#4 ❌ RUBRIC SYSTEM
   Ada di: Canvas, Moodle, Blackboard
   Cendekia: TIDAK ADA
   
   Contoh di Canvas:
   - Assignment linked ke rubric
   - Criteria: "Creativity (5pts), Accuracy (3pts), Presentation (2pts)"
   - Performance levels: "Excellent / Good / Fair / Poor"
   - Comments template per criteria
   - Auto-calculate score
   
   Cendekia: Just "good job" + input 85. That's it.
   
   IMPACT: Grading subjective.
           Students ga tahu exact ekspektasi apa.
   
   Timeline: 1-2 minggu

─────────────────────────────────────────────────────────────────────────────

#5 ❌ ANALYTICS / LEARNING ANALYTICS
   Ada di: Canvas, Moodle
   Cendekia: TIDAK ADA
   
   Contoh di Canvas:
   - Student engagement: "John spent 2hrs in course, accessed 5 pages"
   - At-risk identification: "Jane hasn't submitted in 3 weeks!"
   - Time spent analytics
   - Content view patterns
   - Grade distribution chart
   
   Cendekia: Just... grade list. Nothing else.
   
   IMPACT: Teachers ga bisa identify struggling students.
           No proactive intervention.
   
   Timeline: 2-3 minggu

─────────────────────────────────────────────────────────────────────────────

#6 ❌ DIRECT MESSAGING / INBOX
   Ada di: Canvas, Blackboard, Moodle
   Cendekia: TIDAK ADA (forum only)
   
   Contoh di Canvas:
   - Private message teacher
   - Compose to multiple people
   - Notification when new message
   - Folders & search
   
   Cendekia: Ga ada. Forum aja. Bukan private.
   
   IMPACT: Students harus tanya di forum (public, ga privacy).
   
   Timeline: 1 minggu

─────────────────────────────────────────────────────────────────────────────

#7 ❌ SUBMISSION TRACKING + RE-ATTEMPT
   Ada di: Canvas, Moodle
   Cendekia: Current submission only
   
   Contoh di Canvas:
   - Submission history: View all past versions
   - Late submission flag
   - Allow re-submission dengan "Keep Best Grade"
   - Detailed audit trail
   
   Cendekia: Only current. No history.
   
   IMPACT: Teachers ga bisa liat who submitted late.
           No re-attempt encouragement.
   
   Timeline: 1 minggu

   - Conditional access

#12 ❌ NOTIFICATION CUSTOMIZATION
   - Email notifications per event
   - In-app notifications
   - Do Not Disturb settings
   - Digest emails
